# Mastodon Archive Viewer

一个纯前端的 Mastodon 存档查看器，数据存储在浏览器本地，无需后端服务器。

## 特性

✨ **一次上传，永久保存** - 数据持久化存储在浏览器 IndexedDB
🔒 **完全隐私** - 数据不离开你的设备
⚡ **快速搜索** - 全文搜索、标签过滤、时间筛选
📱 **响应式设计** - 支持桌面和移动设备
💰 **完全免费** - 无服务器成本，静态托管
📴 **离线可用** - PWA 支持，无网络也能访问

## 问题解决

现有 Mastodon 存档查看工具的痛点：
- ❌ 每次都要重新上传
- ❌ 刷新页面数据丢失
- ❌ 没有远程持久化

本项目的解决方案：
- ✅ 上传一次，数据永久保存在浏览器
- ✅ 刷新页面数据依然存在
- ✅ 无需服务器，完全本地化

## 技术栈

```
前端: React + TypeScript + Vite
样式: Tailwind CSS
存储: IndexedDB (Dexie.js)
搜索: Fuse.js
部署: Vercel / GitHub Pages (免费)
```

## 核心思路

**洞察**：Mastodon 存档本身就是 JSON 文件，且是只读数据，完全不需要传统数据库！

**方案**：
1. 用户上传 ZIP 存档
2. 浏览器内解析 JSON 和媒体文件
3. 存入 IndexedDB（浏览器本地数据库）
4. 前端直接查询和展示
5. 数据持久保存，刷新不丢失

```bash
npm run build
```

## 如何使用

1. 从 Mastodon 导出你的存档（设置 → 导入和导出 → 申请存档）
2. 下载 ZIP 文件
3. 访问本应用，上传 ZIP 文件
4. 浏览、搜索你的历史帖子
